# Heraldradio
Heraldradio.com player for the RPi<br>
Install latest raspbian image<br>
<p>
then
<p>
#Before git
<br>
sudo apt-get install git <br>
git clone git://github.com/llrepairs/heraldradio.git <br>
<p>

#After Git<br>

cd heraldradio <br>
sudo chmod +x install.sh <br>
sudo ./install.sh <p>

Thats it.
